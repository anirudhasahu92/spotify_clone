
# Spotify Clone – Concept Project

A front-end concept project inspired by Spotify’s UI, built using pure HTML, CSS, and JavaScript.  
This project demonstrates a simple music player interface with a playlist, play/pause controls, progress bar, and basic navigation.

## Preview

The project includes:
- Top navigation bar with branding and menu  
- Playlist section with song covers and durations  
- Play/Pause, Previous, Next controls  
- Progress bar with current playing track display  
- Background hero section with music-themed artwork  

---

## Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)

## Features

- Playlist with multiple songs  
- Play / Pause toggle

## How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/anirudhasahu92/spotify_clone.git
   ``` 

3. Open the project folder:
   ```bash
   cd spotify_clone
   ```

5. Open index.html in your browser:
   - Double click the file  
   - Or use VS Code Live Server
